Правила по миграции базы данных

1. Скачать последнюю(18) версию PostgreSQL с официального сайта(https://www.postgresql.org/download/)
2. При установке ставим пароль: vibecore (это важно!). Всё остальное по умолчанию.
3. находим новый скачанный SQL Shell (psql).
4. нажимаем ENTER до тех пор пока не увими надпись Password for user postgres(возможно будет на русском).
5. вбиваем пороль из пункта 2 (вы не будете видеть, что вбиваете, поэтому пишите без ошибок).
6. если всё прошло удачто пишем: CREATE DATABASE vibecore;
7. далее заходим по пути репозитория db/migrations/1_create_table_users.sql и в консоли vs code пишем 
   команду: & "C:\Program Files\PostgreSQL\18\bin\psql.exe" -h localhost -U postgres -d vibecore -f D:\web_client\migrations\1_create_table_users.sql 
   (если путь другой, то поменяйте или попробуйте использовать "psql" в место него)
8. а вообще лучше сразу в дс идти, просто для справки написал, чтобы самому не забыть.
